# AGENTS.md

## Project Goal
Build a web application to manage a golf league with:
- Seasons
- Players
- Weekly rounds or matches
- Score entry (gross + net)
- Handicaps
- Standings and leaderboards
- Admin vs player permissions
- Mobile-friendly UI

## Tech Stack
- Next.js (App Router)
- Prisma ORM
- Postgres database
- Clean, readable React components

## Rules for Codex
- Prefer simple, explicit solutions
- Avoid over-engineering
- Ask before major refactors
- Ensure the app runs locally after changes
- Run lint when appropriate
